This repository contains database design projects that can be used for real applications. In addition to the design work for learning purposes, I have implemented these projects using raw database syntax — SQL for relational databases and native queries for NoSQL systems. To make working with the database easier, I’ve also included common features such as views, functions, triggers, transactions, and stored procedures. Seed data is provided as well, along with use case queries that demonstrate how the database supports its requirements.

Each project includes:
- **System Requirements** 
- **ERD Files** – diagrams showing entities, relationships, attributes, and keys. Each design is reviewed for correctness and normalized to at least 3NF.
- **Implementation Scripts** – raw SQL or NoSQL scripts that implement the ERD, including indexing strategies, attribute metadata, and advanced features like transactions, views, access control, and more.
